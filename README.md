# NeuralDigitRecognizer

Полносвязная нейронная сеть, реализованная с нуля на C# без использования ML-фреймворков.

Проект включает:
- реализацию алгоритма прямого распространения (Forward Propagation),
- обратного распространения ошибки (Backpropagation),
- обучение на датасете MNIST,
- WinForms-приложение для распознавания рукописных цифр.

## Технологии

- C# (.NET 8)
- WinForms

## Архитектура сети

Конфигурация:
- 784 входа (28x28)
- 128 нейронов в первом скрытом слое
- 64 нейрона во втором скрытом слое
- 10 выходов (классы 0–9)

Активационная функция:
- Sigmoid

Функция ошибки:
- Mean Squared Error (MSE)

Обновление весов:
- градиентный спуск
- mini-batch обучение

## Реализованные алгоритмы

- Инициализация весов
- Прямое распространение сигнала
- Вычисление градиентов по методу backpropagation
- Обновление весов через learning rate
- Сохранение и загрузка весов из файла
- Нормализация входных данных (0–1)

## Обучение

- Датасет: MNIST (CSV формат)
- Размер входного вектора: 784
- One-hot кодирование выходных классов
- Случайная выборка mini-batch при обучении

После обучения веса сохраняются в файл `trained_weights.txt`.

## Desktop-приложение

WinForms-интерфейс позволяет:

- рисовать цифру на холсте 28x28
- выполнять инференс обученной модели
- выводить предсказанный класс

Модель загружается из сохранённых весов.
